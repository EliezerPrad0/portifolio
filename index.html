Conhecimentos

HTML

HTML é a linguagem de marcação universal em uso na Web. HTML permite que você formate o texto, adicionar imagens, criar links, formulários de entrada, quadros e tabelas, etc.

CSS

CSS é uma "folha de estilo" composta por “camadas” e utilizada para definir a apresentação em páginas da internet que adotam para o seu desenvolvimento linguagens de marcação como o HTML.

React

React é uma biblioteca de JavaScript, open source, utilizada para construir user interfaces, nomeadas para aplicações de página única. Assim, possibilita aos desenvolvedores criarem aplicações web, tendo a facilidade de alterar elementos, sem recarregar a página.

Javascript

Javascript é uma liguagem de programação interpretada estruturada, de script em alto nivel com tipagem dinâmica fraca e multiparadigma

Booststrap

Booststrap é um framework Front End. A partir dele, muitas etapas do desenvolvimentos web se tornam mais rápidas e dinâmicas, pois já trazem consigo diversos elementos prontos e estilizados.

Git e GitHub

Git é um software de controle de versão de projetos para desenvolvedores.
Já o Github é a "Rede Social" dos dev onde você pode colocar esses projetos versionados
